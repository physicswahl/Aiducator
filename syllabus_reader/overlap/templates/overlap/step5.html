Game Analysis & Learning
Key Lessons Learned
A Pattern Is Anything You Can Recognise
Our (simple) pattern was a circle on a fixed place on a screen.
You Can Measure How Well Two Patterns Match
For our circle game, we used the area of overlap between the our circle and a hidden circle.
If You Can Measure It You Can Improve It
Once we had our measure, we could figure out how to do better, especially if we knew the direction which improved our measure.

// Algorithm to find optimal circle placement for maximum overlap
function findOptimalOverlap(targetArea, circleRadius) {
    bestScore = 0
    bestPosition = null
    // Grid search approach - test multiple positions
    for (x = 0; x < canvasWidth; x += stepSize) {
        for (y = 0; y < canvasHeight; y += stepSize) {
            // Calculate overlap score for this position
            overlapScore = calculateOverlap(x, y, circleRadius, targetArea)
            // Keep track of best position found so far
            if (overlapScore > bestScore) {
                bestScore = overlapScore
                bestPosition = {x: x, y: y}
            }
        }
    }
}
