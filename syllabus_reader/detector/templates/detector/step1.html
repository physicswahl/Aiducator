CO₂ Detector Setup & Simulation

In this step, you will learn about CO₂ detection in classroom environments:

  • Run the simulation: Start the CO₂ monitoring simulation to see how levels change over 30 minutes
  • Observe patterns: Watch how CO₂ gradually increases from 400 PPM to 700 PPM as a classroom fills up
  • Test spike detection: Use the "Trigger Spike" button to simulate sudden CO₂ increases (e.g., when many people enter)
  • Configure monitoring: Choose detection mode and set alert thresholds
  • Define objectives: Specify what you want to monitor and why
Tip: Normal outdoor CO₂ is ~400 PPM. Indoor levels of 700-800 PPM indicate poor ventilation, while levels above 1000 PPM can cause drowsiness and reduced cognitive function.
